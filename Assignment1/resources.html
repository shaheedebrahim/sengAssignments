Resources

Quintessential Books

  • Competitive Programming 3, this book is the number 1 book that almost all top level competitive programmers own. It goes over each and every single algorithm that you would ever need in a competition with code examples. Note that most of the book is written in C++ with a few examples in Java.
  • Competitive Programmers Handbook, this book is one that was created by an international master on Codeforces named 'pllk'. It is mostly geared towards the International Olympiad in Informatics (IOI).
  • Introduction to Algorithms is one of the most well known algorithms books, it covers a lot of topics past what you would need to know for competitive programming, however a solid understanding of the fundamentals in this book will lead to success

Algorithm and Data Structure Resources

  • Coursera hosts a wide variety of algorithms courses that can be taken from the begginner to advanced level
  • VisuAlgo is a fantastic resource for visualising what algorithms look like instead of just reading the code. If you are having a hard time understanding what an algorithm actually does this is a great resource
  • Algo.is is a website created by Bjarki Ágúst Guðmundsson developer for the online judge Kattis and word ACM-ICPC finalist. It segments Kattis problems into their specific topics so students can train on a specific algorithm type.

Competitive Programming Networks

  • University Competitive Programming Clubs are found at almost every single university. These are great networks as it is much easier to learn with experienced peers that by yourself
  • Codeforces Groups These groups are a great way to join with others of the same level or higher to enhance your ability
